## Example 1: Simple Linear Docs

**Use case:** Small documentation site with sequential learning path.

```typescript

  docs: {
    defaultTab: {
      label: "Docs",
      icon: "book"
    },
    groups: [
      {
        id: "main",
        label: "Documentation",
        autoGenerated: true,
        entries: [
          { slug: "introduction" },
          { slug: "installation" },
          { slug: "getting-started" }
        ]
      }
    ]
  }
}
```
