# Form Component

Build complex forms with ease.

## Basic Usage

```tsx
<Form onSubmit={handleSubmit}>
  <Input name="email" type="email" />
  <Button type="submit">Submit</Button>
</Form>
```

## Features

- Built-in validation
- Error handling
- Accessible by default
