Welcome! We're excited you're here. CelestialDocs is a modern documentation template built with [Astro](https://astro.build) that makes creating beautiful, organized documentation a breeze.

## What is CelestialDocs?

CelestialDocs is a flexible documentation template designed to help you create professional documentation sites quickly. It combines the power of Astro's static site generation with an intuitive navigation system that scales from simple single-page docs to complex multi-collection documentation sites.

Think of it as your documentation's best friend - it handles the structure, navigation, and styling so you can focus on writing great content.

### üéØ Flexible Navigation

CelestialDocs offers multiple ways to organize your content:

- **Groups** - Organize related pages together in the sidebar
- **Tabs** - Create distinct sections for different topics
- **Auto-generation** - Let the system discover and organize files automatically
- **Manual control** - Explicitly order pages when you need precise control

Mix and match these approaches to create the perfect structure for your documentation.

CelestialDocs is perfect for:

## What Makes It Different?

Unlike other documentation tools, CelestialDocs gives you:

1. **True multi-collection support** - Not just multiple sections, but completely separate documentation sites in one project
2. **Flexible navigation** - Mix manual and auto-generated navigation in the same sidebar
3. **Tab-level organization** - Create distinct navigation contexts without overwhelming users
4. **Simple configuration** - Everything is configured in one TypeScript file with full type safety
