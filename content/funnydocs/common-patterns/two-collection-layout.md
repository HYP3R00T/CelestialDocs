Often you want two separate documentation systems side-by-side. Also, sometimes you want two separate joke streams.

- **Docs** - User guides, tutorials, getting started
- **API** - API reference, endpoints, types

## Setup (serious steps, silly comments)

### Step 1: Create Folders

```bash
mkdir -p content/docs
mkdir -p content/api
```

### Step 2: Configure in data/config.ts

```typescript
export const CONTENT = {
  systems: [
    {
      id: "docs",
      dir: "content/docs",

      route: "/docs"
    },
    {
      id: "api",
      dir: "content/api",
      defaultDocRedirect: "/api/reference",
      route: "/api"
    }
  ]
}
```
